# PostgreSQL Runtime Folder

ضع هنا نسخة PostgreSQL **runtime-only / portable binaries** المستخدمة للتشغيل المحلي.

أمثلة الملفات المتوقعة لاحقًا:
- `bin/postgres.exe`
- `bin/pg_ctl.exe`
- `bin/pg_isready.exe`
- `bin/createdb.exe`
- `bin/pg_dump.exe`
- `bin/psql.exe`

> ملاحظة: لا نضيف binaries الفعلية إلى Git.

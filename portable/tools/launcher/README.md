# ZS Portable Launcher

This folder is organized so non-technical users can find the main actions quickly.

## Main entrypoints (use these)
- `Start-ZS.bat`: starts ZS Portable.
- `Stop-ZS.bat`: stops ZS Portable and frees the folder for move/delete.
- `Setup-ZS.bat`: runs first-time setup.
- `Diagnostics-ZS.bat`: generates diagnostics output.

## Technical subfolders
- `scripts/`: core PowerShell launcher implementations.
- `shortcuts/`: desktop/startup shortcut install/remove scripts.
- `lib/`: shared helper scripts.
- `server/`: frontend static server runtime script.

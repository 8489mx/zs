# خطة إعادة بناء الموارد البشرية

## الهدف
إعادة بناء موديول الموارد البشرية بشكل نظيف ومناسب للشركات الصغيرة والمتوسطة في مصر، بعد حذف الواجهة القديمة غير المعتمدة.

## مراحل التنفيذ
1. التأسيس
   - جرد الـ API + تثبيت المصطلحات + طبقة service + تنظيف الراوتس
2. الأساسيات
   - الموظفين + إضافة موظف + ملف الموظف + العقد + المستندات
   - الرقم القومي حقل اختياري في إضافة الموظف، ومهم لاستكمال ملف الموظف والامتثال داخل مصر.
3. التشغيل اليومي
   - الحضور والانصراف + الإجازات + الشيفتات + العطلات الرسمية
4. البنود المالية المتغيرة
   - السلف + الخصومات + الإضافات + المكافآت
5. المرتبات
   - كشف المرتبات + مراجعة + اعتماد + قفل + مفردات مرتب + تصدير
6. التقوية
   - التقارير + الصلاحيات + Audit + الاستيراد والتصدير

## مبادئ التصميم
- ممنوع إنشاء صفحة HR ضخمة أو monolithic.
- ملف الموظف هو مركز الموارد البشرية.
- المرتبات تأتي بعد الموظف، العقد، الحضور، الإجازات، والسلف والخصومات.
- السياسات والإعدادات التي تتغير بمرور الوقت يجب أن تدعم تواريخ سريان.
- الموافقات والـ Audit طبقات مشتركة في العمليات الحساسة.
- الاستيراد والتصدير CSV/Excel مهم قبل التكاملات المعقدة.
- الواجهة يجب أن تكون متناسقة مع باقي المشروع.

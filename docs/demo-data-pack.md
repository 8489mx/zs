# Demo data pack

هذه الإضافة توفر مسارًا محليًا سريعًا لتجهيز قاعدة بيانات قابلة للتجربة بدل البدء من قاعدة فارغة.

## Available commands
- `npm run seed:demo`
- `npm run reset:demo`

## Seed behavior
- يرفض العمل إذا كانت القاعدة تحتوي بيانات تشغيلية موجودة مسبقًا، إلا إذا استخدمت `--force`.
- يدعم `DB_FILE` أو `--db-file` لاختبار seed على قاعدة بيانات مؤقتة.
- يكتب تقريرًا بعد التنفيذ إلى `docs/demo-seed-report.json`.

## Seeded scenarios
- شراء نقدي وآجل
- بيع نقدي وآجل
- تحصيل عميل
- سداد مورد
- مرتجع بيع
- مرتجع شراء
- مصروفات خزينة
- بيانات وردية كاشير مغلقة
- تحويل مخزني
- جلسة جرد مع فرق
- سجل تالف

## Recommended local workflow
1. `npm run reset:demo`
2. `npm run dev`
3. الدخول بالحسابات الثلاثة والتحقق من الصلاحيات والـ branch scoping
4. تنفيذ UAT محلي على نفس البيانات بدل إدخال كل شيء يدويًا

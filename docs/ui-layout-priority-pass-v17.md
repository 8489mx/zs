# UI layout priority pass v17

## What changed
- Accounts now opens ledgers before payment forms, so review starts from balances and register history first.
- Treasury now opens transactions and expenses registers before expense entry, so the operator sees cash movement before creating a new expense.
- Sales now opens the sales register before guidance cards, so review and selection start immediately from the register.
- Inventory overview now opens with stock status and search before action panels, so the operator sees stock state first.

## Intent
Show the current operational state first on monitoring and review screens, then show creation or secondary actions.
